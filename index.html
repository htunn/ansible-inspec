🚀 Ansible-InSpec Helm Charts Repository

Enterprise compliance testing platform combining Ansible and InSpec with Kubernetes deployment support.

📦 Artifact Hub 💻 GitHub 🐳 Docker Hub 🐍 PyPI

📋 Add This Repository

helm repo add ansible-inspec https://htunn.github.io/ansible-inspec/helm
helm repo update

🔍 Search for Charts

helm search repo ansible-inspec

💾 Install Chart

helm install my-release ansible-inspec/ansible-inspec

⚙️ Install with Custom Values

helm install my-release ansible-inspec/ansible-inspec \
  --set postgresql.enabled=true \
  --set postgresql.auth.password=yourpassword \
  --set image.tag=0.2.6
